CHAPTER 4: TEXT CLASSIFICATION

We explore how large language models can be used for text classification. We cover both representation models (encoder-based) and generative models (text-to-text).

🧠
1. What is Text Classification?
Text classification = assigning a text to one or more predefined labels.
Examples:
  • Sentiment analysis (positive/negative)
  • Topic categorization (sports, politics)
  • Spam detection (spam / not spam)
  • Intent detection (FAQ bots)
  • Toxic content detection
🎯 Goal:
Map a piece of text → the correct label(s).

7. Using a Task-Specific Model
Domain-specific models are trained for a particular task or domain.
Example: Sentiment model trained on Amazon reviews.
How to use:
  1. 1Find a model on Hugging Face Hub (e.g., cardiffnlp/twitter-roberta-base-sentiment-latest)
  2. 2Tokenize your text
  3. 3Get predictions directly
Benefits:
  • Better accuracy on specific domain
  • Less data needed
  • Faster to deploy

12. Evaluation Metrics
  • Accuracy = correct / total
  • Precision = TP / (TP + FP)
  • Recall = TP / (TP + FN)
  • F1-score = 2 × (P × R) / (P + R)

13. Practical Python Example (Representation Model)
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

model_name = "bert-base-uncased"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForSequenceClassification.from_pretrained(model_name)

text = "This movie was fantastic!"
inputs = tokenizer(text, return_tensors="pt",
                   truncation=True, padding=True)

outputs = model(**inputs)
probs = torch.softmax(outputs.logits, dim=-1)
print(probs)
# tensor([[0.0023, 0.9977]])
# [neg, pos]
14. Practical Python Example (Generative Model – FLAN-T5)
from transformers import AutoTokenizer, AutoModelForSeq2SeqLM

model_name = "google/flan-t5-base"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForSeq2SeqLM.from_pretrained(model_name)

prompt = """Classify the sentiment of the following
review as Positive, Negative, or Neutral.
Review: I hated this movie.\nSentiment:"""

inputs = tokenizer(prompt, return_tensors="pt")
outputs = model.generate(**inputs, max_new_tokens=5)
result = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(result)
# Negative

16. Key Takeaways
  • Use encoder models for high accuracy with labeled data.
  • Use generative models for flexibility & zero-shot.
  • Start with strong baselines like BERT-base.
  • Prompts matter in generative classification.
  • Evaluate carefully and iterate!
